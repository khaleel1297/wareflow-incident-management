# WareFlow Incident Management Prototype

A simple AI-assisted incident management system designed for support operations and incident visibility.

## Features

- Incident raising portal
- Short ticket IDs
- SLA mapping
- Support dashboard
- AI-assisted incident categorization
- Team assignment
- Email alert notifications

## Technologies Used

- Python
- FastAPI
- SQLite
- HTML/CSS/JavaScript
- Gmail SMTP

## Purpose

This project was built as a support operations automation prototype to improve incident handling and visibility.
